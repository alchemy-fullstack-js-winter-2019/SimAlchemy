## Getting Started
1. Clone and download [GitHub repo](https://github.com/team-sailboat/SimAlchemy)
1. Install dependencies:\
`npm i`

3. Run scripts:\
`npm run test`\
`npm run test:watch`\
`npm run lint`\
`npm run pretest`\
`npm run start` (start server)
